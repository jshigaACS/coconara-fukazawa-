ツール概要説明

このツールは商品コード（ReferenceID, UPC-code, EAN-codeなど）から価格の安い順に商品情報を取得するツールです。
以下の手順に従って操作してください。

  • 1. 商品コードを入力する（1つでも複数でも可能）
  • 2. あなたのアプリIDを入力する
    (eBay API Developes Programより事前に取得してください)
  • 3. 商品検索ボタンを押す
  • 4. 検索結果がCSVファイルでダウンロードされます。
  • 5. ファイルをエクセルで開いてください
